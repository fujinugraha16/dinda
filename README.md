# Happy Birthday, Dinda

An elegant birthday congratulations page built with Next.js 16.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customize the Message

Edit the birthday copy in [`src/content/birthday.ts`](src/content/birthday.ts):

- `recipientName` — the birthday person's name
- `heroSubtitle` — subtitle under the main heading
- `messageParagraphs` — heartfelt message paragraphs
- `wishes` — list of birthday wishes
- `closingLine` — closing sign-off

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
